🚀 HireHub - Job Portal Web Application
HireHub is a full-stack job portal web application that connects job seekers with companies. Built with the MERN stack, it provides a seamless experience for users to browse jobs, apply for positions, and for companies to manage their listings and applicants.

🌟 Features
👤 For Job Seekers
Register & Login with secure authentication
Browse and search jobs with filters
Apply for jobs directly from the platform
View and update your profile
Track all applied jobs

🏢 For Companies
Company registration and management
Post and manage job listings
View all applicants for each job
Update applicant status (Accept / Reject)
Filter and search applicants

🛡️ Admin
Admin dashboard to manage companies and jobs
Filter company and job listings
Full control over platform content
